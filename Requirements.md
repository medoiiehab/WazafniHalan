# Job Requirements Document for the Development and Improvement of the "Waffny Hala" Website

## 1. Home Page Modifications
Improve and modify the design of the home page to increase visibility and user interaction.
- Optimize the home page for search engines (SEO).
- Optmize the home page for mobile devices.

## 2. Appearance on the Google Jobs Board
- Add a **Unique Job Title** for each job.
- Write a **complete and clear job description**.
- Specify **requirements and qualifications** in an organized manner.
- Clarify the **application method** (link, WhatsApp, email).
- Adhere to **Structured Data** for jobs.

## 3. Adding an FAQ Section
- Create a dedicated FAQ section within job pages.
- Use **FAQ Schema** to improve visibility in search results.

## 4. Advertisements on Job Pages and Blog Articles
- Add an advertisement after the first **H1 heading**.
- Add an advertisement in the middle of the content.
- Add an advertisement before the application method section.
- Distribute advertisements without affecting the user experience.

## 5. Improving Website Speed (Especially on Mobile)
- Enhance site speed on mobile devices.
- Compress images and enable **Lazy Load**.
- Improve **Core Web Vitals** (LCP, CLS, INP).
- Reduce unnecessary **CSS** and **JavaScript** files.

## 6. Modifying Rank Math Settings
Adjust Rank Math settings for:
- Blog
- Job pages
- Activate the appropriate **Schema** for each page type.
- Improve default titles and descriptions.

## 7. SEO Issues in the Blog
- The blog currently lacks **Meta Descriptions**.
- Add automatic and manual meta descriptions for each article.
- Improve the indexing of blog articles.

## 8. Required Additions to the Dashboard
- Display the number of jobs per author.
- Show statistics on the number of jobs published monthly.
- Allow saving jobs or articles as **drafts**.
- Add new authors only through the admin.
- Define clear permissions for authors.

---

This document outlines the essential requirements for improving the website technically, in terms of SEO, and administratively, with the goal of increasing visibility in search results and the jobs board, and enhancing the user experience.
